## âœï¸ How to Edit

### Change Text Content
Open any `.html` file in a text editor (Notepad, VS Code, etc.) and modify the text between the HTML tags.

### Change Colors
Open `css/styles.css` and modify these variables at the top:
```css
:root {
  --bg-dark: #121212;        /* Background color */
  --accent-red: #ff4757;     /* Red accent color */
  --text-light: #f5f5f5;     /* Text color */
}
```

### Add Your Photo
1. Replace `images/profile-placeholder.jpg` with your photo
2. Keep the same filename, or update it in `index.html`

### Make Contact Form Work
1. Sign up free at [Formspree.io](https://formspree.io)
2. Create a new form and get your Form ID
3. In `contact.html`, replace `YOUR_FORM_ID` with your actual ID
